Introduction

Using a dataset of LinkedIn job postings from 2024, I will highlight the 20 most common job skills listed for both a Data Scientist and Machine Learning Engineer. With this dataset, I will use Social Network Analysis (SNA) to identify which job skills are most frequently connected to another job skill for each role (high edge weights), which of these job skills have the most strong and frequent connections relative to all other job skills (high weighted degree), and which job skills are more specialized, yet likely critical to the role when listed in the job description (high betweenness). With this information, companies can make more strategic hiring decisions by identifying candidates with specialized, impactful skills, while job seekers can better align their interests and expertise to meet the specific needs of potential employers.

 

Data Cleaning

After filtering out all other job postings, the data set returned 865 jobs Data Scientist roles and 862 Machine Learning roles. Next, I separated each of job skills listed in each job posting and combined similar skills to avoid redundancy. For example, if a job skill on a particular posting data analysis was listed and another skill listed was data analytics , I combined these into one skill called data analytics . After this I summed all of the job descriptions together and grouped them by job description (either Data Scientist or Machine Learning). I did this to find the most frequently included skills within each job title. In the interest of keeping the analysis general along with avoiding visual complexity of each Social Network Analysis plot, I reduced the total number of job skills to the most frequent 20 skills for each position. Find Specialized and Important Skill Sets Within Each of the Roles

Betweenness can be defined as a measure of how important the node is to the flow of information through a network. Betweenness in this instance seems to represent job skills that may be particularly important for an individual job listing, but may not occur in the most job postings. Below are plots where the node size is based on the betweenness of each job skill.

 

 

Java, Pytorch, and Natural Language Processing (NLP) all have a high betweenness score. This may represent a specialized skill within the Data Scientist role that a particular company is looking for. For example, if the job description contains Natural Language Processing, a company may especially be looking for a candidate to be able to analyze and process text data. If Pytorch is listed on the job description, this may indicate a particular need to be able to build and deploy machine learning models.

 

In machine learning positions, Pandas, Kubernetes, and data preparation represent high-betweenness job skills. In this instance, Pandas and data preparation may represent a company looking for an emphasis on the use of pandas for data cleaning, data transformation, feature engineering, etc. in preparation for model building or Kubernetes for model deployment.

 

Conclusion

To conclude, using Social Network Analysis with data from 2024 LinkedIn job postings, we found that important core skills of a Data Scientist include the use of Python, R, SQL, Data Visualization and Data Analytics. In addition to these core skills, Java, Pytorch, NLP seem to be specialized skills within the Data Scientist job field companies are looking for.

 

Whereas in Machine Learning careers core competencies include Python, Tensorflow, Pytorch, AWS, and Azure. In addition to this, an understanding of data preparation needed for model building and utilization of Kubernetes for model deployment are specialized skills companies may be looking for.

 

Understanding these high-betweenness skills can help companies make strategic hiring decisions and allow job seekers to better align their interests and skills to what a company is specifically looking for.
